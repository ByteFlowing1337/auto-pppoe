#What is **auto-pppoe**?
**auto-pppoe** is a script written in python that helps users to make
DHCP request automatically to change their IP address.

#Why use **auto-pppoe**?
It gives us great convenient to change IP address(if not static),
and it's compatible to various operating systems.

#How to use **auto-pppoe**?
1. Setting environment variables for your router's plane password,
and your PPPOE username and password.
2. Run `python pppoe.py`.

#Tips
Some hard codes of this tool is based on TP-link, and currently it is aiming at 
assigning a spec ASN, so you may need to change some codes by yourself.
